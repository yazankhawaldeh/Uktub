# Uktub
A terminal-based open source text editor, definitely a unique idea that has never been thought of before. 

# What is "Uktub"?

- Uktub is the arabic word for "write", which is why is was chosen for a text editor.

# What needs to be done

- Add more functionality to the text editor. This includes backspace, del, down a line up line, traversing via arrow keys (for now).
- Set up a modal editor to have normal and insert modes (vim inspired).
- Implement navigatable menus to see files and etc.

# System requirements

- This is a Linux first program for now, so a linux distribution is required. If you do not have a Linux distribution, then you can use Github Codespaces to develop and test.
- On most linux distributions, ncurses in pre installed with gcc, it should be inside /usr/include/. However if you do not have the ncurses library, then you can install it with your package manager (apt, pacman, dnf, etc).

# How to run

- Pull the repository
- in your terminal inside the root folder where the Makefile is, run "make". An exectuable called "Uktub" will show up.
- Run ./uktub in terminal
